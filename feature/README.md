# Feature engineering phase

Run the notebooks in this order. Each notebook is a single file with its outputs
committed (logs, tables, plots) — re-run a notebook to refresh its outputs.

| Step | Notebook | What it does |
|------|----------|--------------|
| 003 | `etl/003-feature-etl.ipynb` | Pull raw interactions from the OLTP table, build per-movie / per-user aggregate parquets under `output/`. |
| 004 | `engineer/004-features.ipynb` | Feast point-in-time join + `IDMapper` + sklearn `ColumnTransformer` → `train_features.parquet` / `val_features.parquet` + `idm.json` + `item_metadata_pipeline.dill`. |
| 005 | `engineer/005-negative-sample.ipynb` | Negative sampling → `train_features_neg_df.parquet` / `val_features_neg_df.parquet`. |
| 006 | `engineer/006-prep-item2vec.ipynb` | Per-user movie-id sequences → `train_item_sequence.jsonl` / `val_item_sequence.jsonl` / `batch_sequences_overfit.jsonl`. |

Then continue with the model phase: `models/item2vec/007-train-item2vec.ipynb`
(Ray Tune HP search + MLflow tracking + Evidently — see `docs/README.md` "Path D").

## Rebuilding the source notebooks

The source notebooks are generated by throwaway builders (not part of the
pipeline):

```bash
uv run python feature/etl/_build_003.py
uv run python feature/engineer/_build_engineer.py     # builds 004, 005, 006
```

## Outputs

All artifacts land under `feature/output/` (gitignored except `.gitkeep`):

```
feature/output/
├── movie_rating_stats.parquet        # 003 (Feast movie feature source)
├── user_rating_stats.parquet         # 003 (Feast user feature source)
├── train.parquet / val.parquet       # 003 (interactions)
└── engineer/                         # 004–006
    ├── train_features.parquet / val_features.parquet
    ├── train_features_neg_df.parquet / val_features_neg_df.parquet
    ├── train_item_sequence.jsonl / val_item_sequence.jsonl
    ├── batch_sequences_overfit.jsonl   # first 2 sequences (item2vec overfit sanity)
    ├── idm.json                        # IDMapper (item/user id <-> dense index)
    └── item_metadata_pipeline.dill     # sklearn ColumnTransformer
```
